# Avivo.ai Quick Start Guide

## Setup
1. Sign up for an account.
2. Generate API key.
3. Install SDK.
4. Configure environment variables.

## Using the AI Assistant
- Ask questions about uploaded documents
- Automate workflows
- Integrate with messaging platforms

## Uploading Documents
Supported: TXT, MD, PDF (auto converted)

## Troubleshooting
- Check API key
- Ensure documents are readable
- Contact support if needed
